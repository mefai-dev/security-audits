## 7. Findings

### Finding #1: Ownership Not Renounced (High)

| Attribute | Value |
|-----------|-------|
| **Severity** | High |
| **Type** | Centralization / Access Control |
| **Status** | Open |

**Description:**

Direct RPC call to `owner()` returns `0x89Ec0be9f780e3d65c9802661c0a37E2AB189F30`. This is an EOA (Externally Owned Account), not the zero address.

**On-Chain Verification:**

```
RPC: eth_call
To: 0xbd7909318b9ca4ff140b840f69bb310a785d1095
Data: 0x8da5cb5b (owner selector)
Result: 0x89Ec0be9f780e3d65c9802661c0a37E2AB189F30

Expected for SAFE: 0x0000000000000000000000000000000000000000
```

**Impact:**

Owner retains control over privileged functions. Single point of failure exists.

**Recommendation:**

Call `renounceOwnership()` to set owner to zero address.
